API Documentaion



This is the API documentation for the expenses table. It contains the schema and route end points for the expenses table.

expenses end points is located at the following URL: http://your-server-name/expenses

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /expenses/all # list all expenses
  • GET /expenses/all/{expense_id} # get expenses detail
  • GET /expenses/filter?column={column_name}&value={value} # filter expenses by single column
  • GET /expenses/report?{column_name}={value} # report expenses by single/multiple column using & operator
  • POST /expenses/create # create one expenses
  • POST /expenses/create/list # create multiple expenses
  • PUT /expenses/update/{expense_id} # update expenses by expense_id
  • PUT /expenses/update?column={column_name}&value={value} # update expenses by any column
  • DELETE /expenses/delete/{expense_id} # delete one expenses
  • DELETE /expenses/delete # delete multiple expenses
