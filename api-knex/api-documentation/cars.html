API Documentaion



This is the API documentation for the cars table. It contains the schema and route end points for the cars table.

cars end points is located at the following URL: http://your-server-name/cars

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /cars/all # list all cars
  • GET /cars/all/{car_id} # get cars detail
  • GET /cars/filter?column={column_name}&value={value} # filter cars by single column
  • GET /cars/report?{column_name}={value} # report cars by single/multiple column using & operator
  • POST /cars/create # create one cars
  • POST /cars/create/list # create multiple cars
  • PUT /cars/update/{car_id} # update cars by car_id
  • PUT /cars/update?column={column_name}&value={value} # update cars by any column
  • DELETE /cars/delete/{car_id} # delete one cars
  • DELETE /cars/delete # delete multiple cars
