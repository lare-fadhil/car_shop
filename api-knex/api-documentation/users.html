API Documentaion



This is the API documentation for the users table. It contains the schema and route end points for the users table.

users end points is located at the following URL: http://your-server-name/users

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /users/all # list all users
  • GET /users/all/{user_id} # get users detail
  • GET /users/filter?column={column_name}&value={value} # filter users by single column
  • GET /users/report?{column_name}={value} # report users by single/multiple column using & operator
  • POST /users/create # create one users
  • POST /users/create/list # create multiple users
  • PUT /users/update/{user_id} # update users by user_id
  • PUT /users/update?column={column_name}&value={value} # update users by any column
  • DELETE /users/delete/{user_id} # delete one users
  • DELETE /users/delete # delete multiple users
