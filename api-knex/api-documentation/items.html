API Documentaion



This is the API documentation for the items table. It contains the schema and route end points for the items table.

items end points is located at the following URL: http://your-server-name/items

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /items/all # list all items
  • GET /items/all/{item_id} # get items detail
  • GET /items/filter?column={column_name}&value={value} # filter items by single column
  • GET /items/report?{column_name}={value} # report items by single/multiple column using & operator
  • POST /items/create # create one items
  • POST /items/create/list # create multiple items
  • PUT /items/update/{item_id} # update items by item_id
  • PUT /items/update?column={column_name}&value={value} # update items by any column
  • DELETE /items/delete/{item_id} # delete one items
  • DELETE /items/delete # delete multiple items
