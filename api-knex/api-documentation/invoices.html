API Documentaion



This is the API documentation for the invoices table. It contains the schema and route end points for the invoices table.

invoices end points is located at the following URL: http://your-server-name/invoices

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /invoices/all # list all invoices
  • GET /invoices/all/{invoice_id} # get invoices detail
  • GET /invoices/filter?column={column_name}&value={value} # filter invoices by single column
  • GET /invoices/report?{column_name}={value} # report invoices by single/multiple column using & operator
  • POST /invoices/create # create one invoices
  • POST /invoices/create/list # create multiple invoices
  • PUT /invoices/update/{invoice_id} # update invoices by invoice_id
  • PUT /invoices/update?column={column_name}&value={value} # update invoices by any column
  • DELETE /invoices/delete/{invoice_id} # delete one invoices
  • DELETE /invoices/delete # delete multiple invoices
